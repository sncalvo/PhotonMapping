# Whitted Renderer

## How to run

In your OS of choice run premake5 with the corresponding action

```
./premake5 gmake
./premake5.exe vs2019
./premake5-mac xcode4
```

This will generate in the build folder the project for you to work in your favourite OS (or IDE).

## Tested this with

- [x] Windows 10 (Visual Studio 2019)
- [x] MacOS (XCode)
- [x] MacOS (clang)
